About Me - Tzipi Adler

סיפור חיי

שמי ציפי, או בשם הולדתי צ'יצ'יליה. נולדתי בשנת 1950 בעיר פוקשני שברומניה, בת יחידה, תחת השלטון הקומוניסטי לאחר מלחמת העולם השנייה.

במהלך ילדותי ברומניה, הוריי דאגו שלא אחוש באף מחסור, למרות הקשיים הכלכליים והפחד הרב מהשלטון. בילדותי, לקחתי חלק בתנועת הנוער הקומוניסטית "פיונירים" למרות הגילויים האנטישמיים, והייתי מצטיינת בלימודים.

בגיל 14 וחצי עליתי לארץ עם משפחתי, וגרתי בקריית אתא. בהתחלה חוויתי קשיים רבים בבית הספר, בגלל חוסר ידיעת השפה העברית, אך בסופו של דבר סיימתי את לימודי התיכון בהצלחה. בזמן מלחמת ששת הימים, בשל העובדה שהיו לי אופניים, גויסתי לתפקיד פיקוח על קיום ההאפלה בעיר, כלומר דאגתי שכל הבתים יהיו ללא אורות דלוקים.

בשירותי הצבאי שובצתי לקורס "פקידות אפסנאיות", אך לבסוף, לאחר הטירונות, שירתי בתפקיד של מש"קית סעד (מש"קית ת"ש) בבסיס של חיל התובלה. באחד מימי השישי בשירותי הצבאי, אף על פי שהייתה כוננות בבסיס, בחרתי "לברוח" מהבסיס יחד עם חברותיי ונקלעתי באופן מקרי לטרמפ בו נסע ראובן, בעלי לעתיד. בצורה קומית, ראובן נישק את הכובע שלי אשר שכחתי במכונית וכך הכרנו. לאחר זמן קצר ראובן ביקש את מספר הטלפון שלי, דבר שגרם לנו להפגש, ומכאן הכל הסטוריה. חצי שנה לאחר מכן נישאנו, ונולדו לנו שלושה ילדים - רויטל, נגה ואושרי.

את לימודיי ביצעתי בטכניון, שם סיימתי תואר הנדסאית בכימיה ועבדתי בתחום מספר שנים. לאחר זמן מה ביצעתי הסבה לתחום המחשבים, ועסקתי בתחום זהבשארית חיי עד הפנסיה. במהלך השנים עבדתי באלבית, בבטכניון ובמספר סטארטפים.

כיום אני סבתא גאה לשבעה נכדים מקסימים: נועם, עדן, ליעד, קרן, אורן, דקל ושקד.


עוד קצת עלי

אני אוהבת לרקוד ריקודי עם, לשחק משחקי קופסה ומשחקי חשיבה, לקרוא ספרים, ולטייל בארץ ובעולם.

אדם שהיווה כדמות להערצה עבורי היה צביקה פיק, כך שהמוסיקה שלו השפיעה עלי רבות והייתה חלק משמעותי מאוד בחיי.

Web Development Evolution

3 ההבדלים העיקריים בין Web 1.0, Web 2.0 ו-Web 3.0:

  1. אינטראקטיביות: Web 1.0 הייתה סטטית וחד-כיוונית, כלומר המשתמשים יכלו רק לצרוך תוכן. Web 2.0 הפכה את הרשת לאינטראקטיבית, עם יכולת של המשתמשים ליצור ולשתף תוכן. Web 3.0 מבוססת על טכנולוגיית בלוקצ'יין ומאפשרת אינטראקציה מרובה יותר ושיתוף משאבים דיגיטליים.
  2. מרכזיות: Web 1.0 ו-Web 2.0 היו מרוכזות, כלומר התוכן והשירותים הוספקו על ידי ספקים מרכזיים. Web 3.0 היא מבוזרת ומאפשרת שיתוף משאבים ללא ישות מרכזית.
  3. דרכי תשלום: Web 1.0 ו-Web 2.0 היו בעיקר מבוססות על כלכלה של פרסום. Web 3.0 מבוססת על קריפטוגרפיה וכלכלה צמודה בה המשתמשים יכולים לשלם ולקבל תשלומים באמצעות מטבעות קריפטוגרפיים.

בקצרה, Web 3.0 היא יותר אינטראקטיבית, מבוזרת ומתבססת על טכנולוגיות חדשות כמו בלוקצ'יין, בהשוואה לדורות הרשת הקודמים.

Use Case Example: Travel Planning

Web 1.0: Searching for travel destinations on Web 1.0 would yield static web pages with basic information about popular tourist spots, accommodation options, and travel agencies. Users would have limited interactivity and could only passively consume information.
Web 2.0: In Web 2.0, users can engage in collaborative travel planning through platforms like TripAdvisor or Airbnb. Users can share reviews, recommendations, and personal experiences, contributing to a dynamic and interactive travel community. They have the ability to interact with content, share their opinions, and participate actively in the travel planning process.
Web 3.0: With Web 3.0, decentralized travel platforms powered by blockchain technology could revolutionize the industry. Users could access decentralized marketplaces for booking accommodations and activities, with transparent pricing and reviews stored securely on the blockchain. Smart contracts could automate transactions, ensuring trust and security in peer-to-peer interactions. Additionally, personalized travel recommendations based on user preferences and past travel history could be provided, while ensuring user control over their data and privacy.
